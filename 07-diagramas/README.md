## 📝 Como Adicionar Diagramas
1. Criar diagrama em Draw.io ou Mermaid
2. Exportar como PNG ou SVG
3. Fazer upload na pasta `07-diagrams/images/`
4. Referenciar no README com `![Nome](images/arquivo.png)`
